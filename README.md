# dotfiles

## Setup

### Initialize

```
$ bash bin/brew-init.sh # for homebrew
$ bash bin/apt-init.sh  # for ubuntu
```

### Core

```
$ make core # install
$ make core-clean # uninstall
```

* dotfiles
* vim
* nano
* tmux

### Fish

```
$ make fish # install
$ make update # update
$ make fish-clean # uninstall
```

* fish shell
* peco
* ghq
* z
* omf

### Textlint

```
$ make textlint # install
$ make textlint-clean # uninstall
```

* npm
